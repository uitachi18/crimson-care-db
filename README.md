# Project Documentation

This project is a web application developed using a modern front-end technology stack. It is designed and maintained through both local development practices and the Lovable web-based development environment.

**Technologies Used**
- Vite  
- React with TypeScript  
- Tailwind CSS  
- shadcn-ui  

These tools provide a fast, efficient, and scalable development workflow.
